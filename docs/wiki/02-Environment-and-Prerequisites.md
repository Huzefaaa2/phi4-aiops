# 02 - Environment and Prerequisites

## Tools Required

- Terraform >= 1.8
- Python >= 3.11
- Git
- Windows Server 2022/2025 image/template
- Ollama runtime access on target VM

## Repo Paths You Will Use

- `terraform/azure`
- `terraform/vmware`
- `scripts/windows`
- `src/aiops_copilot`

## Suggested Lab Topology

- 1 x Azure Windows VM
- 1 x VMware Windows VM
- 1 x admin workstation with Terraform + Git

## Secrets and Credentials

Prepare:

- Azure SPN credentials
- vSphere credentials
- Windows local admin passwords
- Optional Teams bot credentials
