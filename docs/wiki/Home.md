## Intended Audience

- Students learning practical AIOps architecture
- DevOps/SRE engineers implementing local AI operations workflows
- Platform engineers deploying on Azure or VMware
- IT operations teams integrating conversational diagnostics in Teams

## Expected Outcomes

By following this guide you will be able to:

- Provision Windows infrastructure with Terraform
- Bootstrap local model + log ingestion services
- Run retrieval-based incident analysis against Windows logs
- Query by server and obtain structured root-cause recommendations
- Integrate a Teams bot pattern for operational workflows
