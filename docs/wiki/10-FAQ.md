# 10 - FAQ

## Is this fully autonomous remediation?
No. Recommended pattern is human-in-the-loop approval.

## Can this run without internet?
Yes, if dependencies and model artifacts are pre-staged internally.

## Why local model instead of cloud LLM?
To improve privacy, latency, and control for operational logs.

## Does this replace monitoring tools?
No. It augments monitoring with reasoning and contextual summaries.

## Can we use this in regulated industries?
Yes, with governance controls, retention policies, and approval workflows.
